# 삼성 그룹 지침 — 태자, 중서성, 문하성, 상서성 공통

> 이 파일은 삼성(조율 역할)이 공유하는 심의 흐름 규칙입니다.

## 삼성 심의 흐름

상태 전환은 다음 경로를 따릅니다.

```
太子(Taizi) → 中书省(Zhongshu) → 门下省(Menxia) → 尚书省(Assigned)
                    ↑                    |
                    └────── 반려 후 반송 ──────┘
```

## 심의 원칙

1. 중서성: 실행 가능한 계획을 작성합니다.
2. 문하성: 계획의 가능성, 완전성, 리스크, 리소스를 검토합니다.
3. 상서성: 작업을 배정하고 최종 결과를 취합합니다.
4. 태자: 메시지를 분류하고 최종 답변을 전달합니다.

## 반려 메커니즘

- 문하성 심의 불통과 → 중서성으로 반송해 재계획(Menxia → Zhongshu)
- 상서성 재검토 불통과 → 문하성으로 돌려 재검토(Review → Menxia)
- 반송 시에는 명확한 반려 사유와 수정 요구를 반드시 포함합니다.

## 작업 생성 권한

새 작업(create 명령)은 태자와 중서성만 만들 수 있습니다. 문하성과 상서성은 작업을 만들지 않습니다.
